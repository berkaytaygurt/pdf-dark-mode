# PDF Dark Mode

A PDF viewer that applies a dark mode effect to improve reading comfort,
especially during long study sessions and night-time use.

## Project Structure
- `web/` contains the core implementation written in HTML, CSS, and JavaScript.
- `android/` contains the Android application that wraps the web version using WebView.

## Android App
This project is published on Google Play as an Android application.

🔗 Google Play: https://play.google.com/store/apps/details?id=com.btayg.pdfdarkmode

## Technologies
- HTML
- CSS
- JavaScript
- Android (WebView)

## Motivation
This project was developed to solve a personal usability problem while reading PDFs
and to gain hands-on experience in building and publishing an application.
